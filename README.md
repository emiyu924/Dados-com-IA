# 📊 Dados com IA — Análise de Dados com Ferramentas de Inteligência Artificial

> Repositório prático para aprendizado de análise de dados, visualização e machine learning usando Python e bibliotecas de IA.

---

## 📋 Sobre

Este repositório reúne um conjunto de notebooks e scripts voltados ao aprendizado prático de **ciência de dados** com ênfase em ferramentas de Inteligência Artificial. O material segue uma sequência didática de aulas, cobrindo desde a manipulação básica de dados até técnicas de machine learning e geração de relatórios analíticos automatizados.

O projeto foi atualizado recentemente (Julho/2026), com adições de gráficos, heatmaps, análises exploratórias e relatórios gerados por IA sobre os dataframes trabalhados.

---

## 🗂️ Estrutura de Arquivos

```
Dados-com-IA/
├── data/                      # Pasta com os conjuntos de dados utilizados
├── app.py                     # Aplicação principal em Python
├── aula02.ipynb               # Manipulação de dados com Pandas
├── aula03.ipynb               # Continuação dos comandos Pandas
├── aula04.ipynb               # Visualização de dados com Seaborn
├── aula06.ipynb               # Gráficos avançados, heatmap e análise de dados
├── aula_03_relatorios.ipynb   # Geração de relatórios analíticos (incluindo IA)
├── bibliotecas.ipynb          # Exercícios com NumPy
├── exercicios_aula02.ipynb    # Exercícios práticos de visualização
└── ml.ipynb                   # Introdução a Machine Learning
```

### Descrição dos Notebooks

| Arquivo | Conteúdo |
|---|---|
| `aula02.ipynb`, `aula03.ipynb` | Comandos no **Pandas**: leitura, filtragem, agrupamento, limpeza e transformação de dataframes. |
| `bibliotecas.ipynb` | Exercícios com **NumPy** para operações numéricas e arrays. |
| `aula04.ipynb`, `exercicios_aula02.ipynb` | Visualização com **Seaborn**: barras, dispersão, histogramas, boxplots. |
| `aula06.ipynb`, `aula_03_relatorios.ipynb` | Análise exploratória avançada: **heatmaps**, correlações, insights e **relatórios por IA**. |
| `ml.ipynb` | **Machine Learning** com scikit-learn: pré-processamento, treinamento e avaliação de modelos. |
| `app.py` | Script de aplicação para execução de análises. |

## 🛠️ Pré-requisitos

- **Python 3.8+**
- Pacotes: `pandas`, `numpy`, `seaborn`, `matplotlib`, `scikit-learn`, `jupyter`

---

## ⚡ Instalação

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/emiyu924/Dados-com-IA.git
   cd Dados-com-IA
   ```

2. **Crie um ambiente virtual (recomendado):**
   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux/Mac
   # ou
   venv\\Scripts\\activate     # Windows
   ```

3. **Instale as dependências:**
   ```bash
   # Se houver requirements.txt:
   pip install -r requirements.txt
   # Caso contrário:
   pip install pandas numpy seaborn matplotlib scikit-learn jupyter
   ```

---

## 🚀 Como Usar

### Notebooks

Abra o Jupyter e execute os notebooks em sequência:

```bash
jupyter notebook
```

Ordem sugerida: `aula02.ipynb` → `aula03.ipynb` → `bibliotecas.ipynb` → `aula04.ipynb` → `exercicios_aula02.ipynb` → `ml.ipynb` → `aula06.ipynb` → `aula_03_relatorios.ipynb`

### Aplicação

```bash
python app.py
```

Os dados estão na pasta `/data`. Ajuste os caminhos nos notebooks se necessário.
